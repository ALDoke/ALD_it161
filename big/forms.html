Social Media Optimization (SMO)

In order to gain any data from the users of a given website; HTML forms are needed in order to collect information like the site visitor’s email address, name, credit card information, or billing address. These are just some common examples for websites that require your billing information to charge your account. These forms will use back-end applications like CGI, ASP Script, or PHP Script in order to take site inputs from users and post them back to the web author by passing the data through a defined database. Some of the HTML forms that are most common include single-line text input controls, password input controls, and multiline text input controls. All of these named text input controls are used to allow site visitors to input their data into these defined fields to feed it back to their databases. In addition, all of these forms can be styled via CSS. Some of the main styling attributes include width control, padding control, margin control, box sizing control, border control, and color control. In addition each of these styling controls can be used to select only specific types of inputs, such as only text inputs rather than all of the inputs on the site. This control of input styles can extend to your entire site similar to most CSS style inputs. In addition CSS allows the author of the website to be able to customize much further than the base HTML, which is used almost exclusively for functionality. CSS can allow for entries to become animated by including transitions, or add images as the background of any input using background image or background position controls. Overall the functionality of HTML forms allows for site users to input information while on the back end the site author can be used to fully style the input parameters and make it the most appealing and easy to use.
